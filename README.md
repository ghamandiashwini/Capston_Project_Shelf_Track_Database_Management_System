# Shelf Track Database Management System

## Overview

Shelf Track Database Management System is a Python and SQLite-based command-line application developed as a capstone project. The system enables users to efficiently manage a library or book inventory by storing, retrieving, updating, and deleting book records from a relational database.

The project demonstrates the use of Python programming, SQL database operations, and CRUD (Create, Read, Update, Delete) functionality to build a practical database management application.

---

## Features

### Book Management
- Add new books to the database
- Search for books by ID or title
- View all books in the database
- Update existing book information
- Delete book records
- Display book details in a structured format

### Database Operations
- Connect to an SQLite database
- Perform CRUD operations
- Execute SQL queries
- Commit changes to the database
- Retrieve and display records efficiently

## How to Run

1. Clone the repository

```bash

```

2. Navigate to the project folder

```bash
cd Shelf-Track-Database
```

3. Run the application

```bash
python bookshelf.py
```
